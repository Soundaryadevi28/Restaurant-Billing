# Restaurant Billing System

A comprehensive Python CLI application designed to automate the ordering and receipt generation process for a multi-cuisine restaurant.

##  Key Features
* **Categorized Menus:** Efficiently manages separate menu datasets for Breakfast, Lunch, and Dinner.
* **Complex Logic Flow:** Utilizes nested conditional branching (`if-elif-else`) to handle diverse user selections.
* **Automated Receipt Generation:** Real-time calculation of total amounts based on item price and quantity.
* **Input Sanitization:** Uses string methods like `.lower()` to ensure a smooth, error-free user experience regardless of input casing.

##  Technical Stack
* **Language:** Python 3.x
* **Core Concepts:** Conditional logic, string manipulation, arithmetic operations, and console-based UI design.

##  Sample Output
The system generates a structured digital receipt including:
- **Item Name**
- **Quantity**
- **Unit Price**
- **Total Payable Amount**
